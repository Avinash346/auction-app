# Real-Time Auction System (MERN + Socket.IO)

## Local setup

### 1) Install dependencies

Server:

```bash
cd server
npm install
```

Client:

```bash
cd client
npm install
```

## Run

### Terminal 1 (server)

```bash
cd server
node index.js
```

Server runs on:

- http://localhost:4000

### Terminal 2 (client)

```bash
cd client
npm run dev
```

Client runs on:

- http://localhost:5173
